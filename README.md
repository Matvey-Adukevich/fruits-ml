# fruits-ml

Цель проекта: обучить нейросеть для определения названия фрукта по его изображению.

Нейросеть реализована на основе нейросети ResNet18

Источники:
1)Датасет для обучения: https://www.kaggle.com/datasets/karimabdulnabi/fruit-classification10-class/data <br>
2)Статья про использование ResNet18: https://www.kaggle.com/code/muhammetcepi/resnet18-classification/notebook <br>
3)Документация библиотеки PyTorch: https://docs.pytorch.org/docs/stable/index.html
